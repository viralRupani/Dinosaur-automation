# Dinosaur-automation


Run Command.
```
1) Fork Repository
2) git clone git@github.com:viralRupani/Dinosaur-automation.git
3) cd Dinosaur-automation
4) python -m venv venv
5) cd venv/scripts
6) activate
7) cd ../..
8) pip install -r requirements.txt
9) open https://elgoog.im/t-rex/ in browser
10) py main.py and change tab to game.
```
